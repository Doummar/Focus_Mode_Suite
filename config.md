# Focus Mode Suite Settings

This add-on allows you to enter a highly focused, distraction-free study environment in Anki.

## Configuration Options

- **shortcut_key**: The keyboard shortcut used to toggle Focus Mode manually (e.g. `F11` or `Ctrl+H`).
- **hide_top_bar**: Hides the menu bar and top decks/add/browse toolbars when Focus Mode is active (`true` or `false`).
- **hide_bottom_bar**: Hides the bottom sync/status bar when Focus Mode is active (`true` or `false`).
- **auto_hide_on_study**: Automatically enter Focus Mode when a study session starts (`true` or `false`).

*Note: You can easily modify these options directly by selecting this add-on from the Add-ons manager in Anki and clicking the 'Config' button!*
